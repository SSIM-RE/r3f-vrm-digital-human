## 经验教训

### 2026-03-18 - DiP 数据格式
- API 返回数据和 npy 文件格式完全相同
- 前端不翻转 Z 轴，与 DiP 原始输出一致

### 2026-03-18 - 手掌方向
- 设置手掌骨骼为无旋转（单位四元数），由小臂决定方向

### 2026-03-02 - VRM 表情机制
- VRM 使用 `+=` 自动累加 morphTargetInfluences
- 直接 setValue 即可，无需手动计算

python -m sample.generate \
  --model_path save/dip/model.pt \
  --autoregressive \
  --guidance_param 7.5 \
