# VRM 虚拟数字人 × OpenClaw

## 项目介绍

这是一个集成 OpenClaw Agent 的 VRM 虚拟形象设计雏形，展示如何将虚拟形象与 AI Agent 结合，实现自然的人机交互。

---

## 功能说明

这个虚拟形象能做什么：

1. **智能对话** - 接入 OpenClaw Agent，理解用户意图并回复
2. **语音输出** - TTS 语音合成，虚拟形象能开口说话
3. **表情变化** - 根据对话情绪改变面部表情（开心、悲伤、惊讶等）
4. **动作配合** - 根据对话内容播放对应动作（挥手、点头等）
5. **口型同步** - 说话时嘴巴自然开合

---

## 设计结构

```
用户输入 (语音/文本)
       │
       ▼
┌─────────────────────────────┐
│      OpenClaw Agent         │
│   (理解意图，返回指令)       │
└──────────────┬──────────────┘
               │
               ▼
┌─────────────────────────────────────────┐
│              前端处理                   │
│  • TTS 语音合成 + 口型同步               │
│  • VRM 表情控制                          │
│  • VRM 动作控制                          │
└─────────────────────────────────────────┘
```

---

## 模块说明

| 模块 | 职责 |
|------|------|
| VRMAvatar | 渲染虚拟形象，控制表情和动作 |
| useVRMControl | 管理虚拟形象的状态（表情渐变、动作切换） |
| useTTS | 语音合成，分析音频驱动口型 |
| useDipMotion | AI 动作生成（可选） |
| useAudioRecorder | 录音功能（带自动截断） |
