# VRM 虚拟数字人 × OpenClaw

## 项目介绍

这是一个集成 OpenClaw Agent 的 VRM 虚拟形象设计雏形，展示如何将虚拟形象与 AI Agent 结合，实现自然的人机交互。

---

## 功能说明

| 功能 | 说明 |
|------|------|
| **智能对话** | 语音/文本输入 → OpenClaw Agent 理解意图 → 返回结构化回复 |
| **语音输出** | TTS 语音合成，虚拟形象同步播放音频 |
| **表情变化** | 根据情绪指令，平滑过渡到对应表情 |
| **动作配合** | 根据动作指令，播放预设动画或 DiP AI 生成动作 |
| **口型同步** | 分析 TTS 音频，实时驱动嘴型开合 |

```

                                 │
        ┌─────────────────────────┴─────────────────────────┐
        │                                                   │
        ▼                                                   ▼
┌───────────────────┐                           ┌───────────────────┐
│  useAudioRecorder │                           │    文本输入      │
│  (录音 + VAD)     │                           │                  │
└────────┬──────────┘                           └────────┬──────────┘
         │                                             │
         └─────────────────────┬───────────────────────┘
                               │

┌─────────────────────────────────────────────────────────────────────────┐
│                         OpenClaw Agent                                  │

│  输出: {voiceText, emotion, actions, expressions}                      │
└────────────────────────────────┬────────────────────────────────────────┘
                                 │

              ┌───────────────────────────────────────────────────────────┐
              │                   前端处理                                │
              │  ┌────────────┐  ┌────────────┐  ┌──────────────────┐  │
              │  │    TTS     │  │   表情     │  │      动作        │  │
              │  │ 语音+口型   │  │  setValue  │  │ 预设动画 / DiP   │  │
              │  └────────────┘  └────────────┘  └──────────────────┘  │
              └────────────────────────┬────────────────────────────────┘
                                       │
                                       ▼
┌─────────────────────────────────────────────────────────────────────────┐
│                          VRMAvatar                                      │
│              (模型加载 + 动画播放 + 表情控制 + 渲染)                    │
└─────────────────────────────────────────────────────────────────────────┘
```

### 模块协作

1. **用户输入** → 录音/文本
2. **Agent 处理** → 理解意图，返回结构化指令
3. **前端解析** → 分发给 TTS、表情、动作模块
4. **VRM 渲染** → 呈现语音、表情、动作

### 动作实现方式

| 方式 | 说明 | 适用场景 |
|------|------|----------|
| **预设动画** | Mixamo 预制 FBX 动画 | 常用动作（挥手、点头等） |
| **DiP 生成** | AI 根据文本生成动作 | 特殊动作、自然效果 |

## 模块说明

### VRMAvatar
- **职责**: 虚拟形象渲染和控制
- **功能**: VRM 加载、骨骼动画、表情控制、眨眼、注视

### useVRMControl
- **职责**: 状态管理
- **功能**: 表情渐变、动作队列、参数配置

### useTTS
- **职责**: 语音合成
- **功能**: TTS 调用、音频分析、口型同步

### useDipMotion
- **职责**: AI 动作生成
- **功能**: DiP 调用、骨骼映射、帧应用

### useAudioRecorder
- **职责**: 录音
- **功能**: 麦克风录音、VAD 自动截断

## 相关技术

### VRM
虚拟形象专用 3D 模型格式，标准化的虚拟形象格式。
- **官网**: https://vrm-c.github.io/

### TTS (语音合成)
将文字转换为语音。本项目使用 GPT-SoVITS。
- **项目**: https://github.com/RVC-Boss/GPT-SoVITS

### DiP (Diffusion Planner)
基于 diffusion 模型的 AI 动作生成技术。
- **项目**: https://github.com/MotionDiffusionModel/MotionDiffusionModel
